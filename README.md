# Blackjack

A personal learning project for getting better at blackjack — both the base game and Hi-Lo card counting. Three modes: **Learn to Play** (tutorial with coached feedback), **Learn to Count** (Hi-Lo tutorial and practice drills), and **Test Yourself** (free play with chips and a running bankroll). Plain HTML/CSS/JS, no build step, deployed as a static site. Live at https://blackjack.milettegillow.com.
